# TF2Lab

TF2Lab - Open Source Competitive Team Fortress 2 Platform

> Helping competitive Team Fortress 2 players improve through data.

🚧 **Work in Progress**

TF2Lab is an open-source platform for competitive Team Fortress 2 players.

Our goal is to provide analytics, performance tracking, competition management and educational resources for the TF2 community.

## Tech Stack

- Nuxt 4
- TypeScript
- Vue 3
- Vercel

## Status

Project is currently under active development.
