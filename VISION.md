# 🎯 TF2Lab - Vision

> **Analyze. Improve. Compete.**

---

# Nossa missão

O TF2Lab nasceu com um objetivo simples:

**Ajudar jogadores de Team Fortress 2 a evoluírem no cenário competitivo através de dados, estatísticas, ferramentas e conteúdo de qualidade.**

Hoje existem excelentes plataformas como logs.tf, demos.tf, trends.tf e more.tf, mas elas focam principalmente em disponibilizar informações.

O TF2Lab pretende transformar essas informações em conhecimento.

Nosso objetivo não é apenas mostrar números.

Nosso objetivo é responder perguntas como:

* Estou evoluindo?
* Em quais mapas jogo melhor?
* Meu desempenho melhorou nos últimos meses?
* Em quais situações estou morrendo mais?
* Meu DPM realmente aumentou?
* Estou jogando melhor do que na temporada passada?
* O que devo treinar?

---

# Problema

O cenário competitivo de Team Fortress 2 possui diversas ferramentas excelentes.

Cada uma resolve uma parte do problema.

* logs.tf armazena estatísticas.
* demos.tf armazena demos.
* trends.tf apresenta gráficos.
* more.tf reúne algumas informações úteis.

Porém nenhuma plataforma reúne tudo em um único lugar.

Além disso, falta uma plataforma voltada para acompanhamento de evolução do jogador.

Também existe pouca documentação organizada para quem deseja começar no competitivo, principalmente na comunidade brasileira.

---

# Nossa visão

Queremos construir a principal plataforma da comunidade competitiva de Team Fortress 2.

Uma plataforma aberta, gratuita e desenvolvida pela comunidade.

O foco não será substituir ferramentas existentes.

O foco será integrar essas ferramentas e oferecer uma experiência melhor para jogadores iniciantes e veteranos.

# Objetivos

## Curto prazo

Construir um dashboard moderno utilizando dados públicos.

Permitir busca por SteamID.

Mostrar estatísticas das partidas.

Criar gráficos de evolução.

Filtros avançados.

---

## Médio prazo

Criar gerenciamento de campeonatos.

Criar gerenciamento de equipes.

Adicionar estatísticas por mapa.

Adicionar comparação entre jogadores.

Criar Academy.

---

## Longo prazo

Criar uma plataforma reconhecida pela comunidade.

Disponibilizar APIs próprias.

Criar integração com Steam Login.

Criar perfis públicos.

Criar sistema de mentoria.

Criar ferramentas para organizadores de campeonatos.

# Filosofia

O TF2Lab não existe para competir com outras plataformas.

Ele existe para conectar todas elas.

Cada serviço já faz um excelente trabalho.

Nossa missão é integrar essas informações e entregar uma experiência melhor para toda a comunidade.

---

# Nossa definição de sucesso

O projeto será considerado um sucesso quando:

* novos jogadores conseguirem entrar no competitivo com facilidade;
* jogadores conseguirem visualizar sua evolução ao longo do tempo;
* líderes de equipe utilizarem o sistema para organizar campeonatos;
* coaches utilizarem os dashboards para revisão de partidas;
* a comunidade brasileira possuir um local central para aprendizado e crescimento.

---

# Nosso compromisso

Este projeto será desenvolvido de forma aberta.

Cada melhoria deverá tornar a comunidade competitiva de Team Fortress 2 mais acessível, organizada e saudável.

O crescimento do projeto será consequência da utilidade que ele entregar para seus usuários.

Nosso objetivo final é simples:

**Ajudar mais pessoas a descobrirem, aprenderem e permanecerem no competitivo de Team Fortress 2.**
